# 🧾 RHC Protocol Core  
**Copyright © 2025 — Fernando Flores Alvarado**

---

### 📖 Descripción

El **RHC Protocol Core** constituye la base técnica y conceptual original del protocolo  
**RHC (Randomized Header Channel)**, diseñado y desarrollado por **Fernando Flores Alvarado**.

Define los principios de **dispersión de encabezados**, **entropía controlada**  
y **autenticación bidireccional**, núcleo del modelo *Controlled Chaos* aplicado  
a la **seguridad web distribuida**.

Este repositorio representa la **versión independiente y fundacional** del protocolo,  
posteriormente compartida con la comunidad **OWASP** bajo el proyecto:  
> *Randomized Header Channel for CSRF Protection*.
